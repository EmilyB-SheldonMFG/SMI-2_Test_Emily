Passcode Locking the Incubator

When the incubator is passcode locked, users are prompted to enter the numeric code before changing the unit settings, including the temperature set point. The code must also be entered to access the Service Functions menu.

 

1. Open the Service Functions menu.

2. Enter 750 in the Special Function Code field.

This brings up the Passcode menu.

3. Enter a 1 to 4-digit numeric passcode.

Once the passcode is set, the user can make changes until the display returns to the homepage, at which point the settings will be locked.

Note: The display will automatically return to the homepage after 120 seconds of inactivity.

 

Passcode Locking, Continued

 

Changing Settings When Passcode Locked

To make changes when a passcode is enabled, enter the passcode on the popup screen. Once the passcode has been entered, the display will bring up the selected settings menu.

The settings will remain unlocked until the display is returned to the homepage.

 

 

Changing the Passcode

Access the Passcode menu and enter a new passcode value. Enter a value of 0 to disable the Passcode Locking function.

 

Invalid Passcode

If an invalid passcode is entered, an alert window will appear. Tap OK to return to the previous screen. There is no limit to the number of times a passcode can be attempted.

 

Note: If the passcode has been lost, please contact Technical Support for assistance.
