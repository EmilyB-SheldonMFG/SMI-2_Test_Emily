Real Time Data Outputs

 

TEMP: 4 – 20mA analog output describing the current chamber temperature.

The output uses a jack port on the right side of the incubator. The port connects to an audio jack (male mono phono plug, ¼ inch (6.3 mm).

Jacks and cables are not included with the incubator.

Parameter Parameter Value at 4mA Parameter Value at 20mA
Temperature 0°C 70°C

 

Data Monitoring Systems – Max Resistance

For building management and other data monitoring or logging systems, the maximum resistance of the current loop from the 4-20mA module is 250 Ohms. At higher loop resistances, the current value will be erroneously low for parameter values near the top of the scale.

 

Alarm: Dry Contact Port

This port communicates all alarm activations as On / Off outputs in which Open position indicates an alarm instance. The port connects to an audio jack and cable (male mono phono plug, ¼ inch (6.3 mm).

Jacks and cables are not included with the incubator.

 

The USB port does not output data in real time.
