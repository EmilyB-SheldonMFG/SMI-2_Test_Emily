Power and Memory

The incubator stores all set points and other settings in non-volatile RAM. The unit will retain its settings indefinitely in the event of a power outage.

 

Data Log

The incubator logs the current chamber temperature and the temperature set point once per minute.

  • Alarm activations are logged at the moment the alarm is activated.
  • Power outages are retroactively logged as starting at the moment of outage.
  • The incubator stores up to 5 years of continually logged data.

Logged data can be viewed as a graph on the incubator by touching the graph icon.

  • The default graph view shows a 24-hour block of time. This is the widest data view.
  • Touching the graph on a vertical guideline centers the graph on that point in time.
  • The graph can be zoomed in to show a 6- or 1-hour block of time.

 

The ( ? ) Help button brings up a color legend for the graph.
